About AI Career Guidance System

AI Career Guidance System is a web-based mini project designed to help students choose suitable career paths based on their interests, skills, personality, work style, and career goals.

Project Purpose

The main purpose of this system is to guide students by providing AI-based career recommendations, compatibility scores, skill suggestions, certifications, salary information, and a career roadmap.

Main Features

👤 User Registration & Login
📊 Student Dashboard
📝 Career Assessment
🤖 AI Recommendation
🥇 Top 3 Career Matches
📄 Download Career Report

Technologies Used

Python, Flask, SQLite, HTML, CSS
