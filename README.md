## рҹҢҹ Overview

Choosing the right career can be challenging. The **AI Career Guidance System** helps students discover suitable career paths by analyzing their **interests, skills, personality, and career goals**.

The system provides personalized career recommendations, compatibility scores, career roadmaps, and downloadable reports through a clean, interactive web interface.

## вҡҷпёҸ Installation

Clone the repository:

```bash
git clone https://github.com/Niranjan3419/AI-Career-Guidance-System.git
```

Move into the project folder:

```bash
cd AI-Career-Guidance-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
