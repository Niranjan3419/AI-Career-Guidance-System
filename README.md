# 🤖 AI-Based Career Guidance System

An AI-powered Career Guidance System developed using **Python, Flask, SQLite, HTML, and CSS**. This application helps students identify suitable career paths based on their interests, skills, work style, career goals, and personality.

---

## 📌 Features

- 👤 User Registration & Login
- 🔒 Secure Authentication
- 📊 Student Dashboard
- 📝 Career Assessment Questionnaire
- 🤖 AI-Based Career Recommendation
- 🥇 Top 3 Career Suggestions
- 📈 Career Compatibility Score
- 💰 Estimated Salary Range
- 🎓 Recommended Courses
- 📚 Certifications
- 🛣️ Career Roadmap
- 📄 Download Career Report

---

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
